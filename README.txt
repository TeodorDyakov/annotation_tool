Съдържание на архива:
annotation_tool папка - съдържа:
    config папка - съдържа конфигурационен файл за базата от данни.
    images папка - съдържа изображения служещи за тестови данни.
    js - Javascript кодът изпозлван в страницата, както и код за интеграция.
    styles папка - css стилове.
    config.php - файл съдържащ информация за проекта.
    db_sql.txt - скриптове за създаване на базата.
    index.php - главната страница на приложението.
    labels_db.php - php код за достъп и заявки до базата относно анотациите.
    documentation.docs - документация на проекта.
